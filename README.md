# Text Converter

This project provides a web-based utility to convert text between TOON and JSON formats.

## How to run

1.  Install the required Python packages:
    ```bash
    pip install -r toon/requirements.txt
    ```

2.  Run the Flask application:
    ```bash
    python app.py
    ```

3.  Open your web browser and navigate to `http://127.0.0.1:5000`.

## How to use

The web interface provides two sections for text conversion:

*   **TOON to JSON**: Paste your TOON formatted text in the left text area and click "Convert to JSON". The result will appear in the right text area.
*   **JSON to TOON**: Paste your JSON formatted text in the right text area and click "Convert to TOON". The result will appear in the left text area.
